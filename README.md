# holbertonschool-web_back_end

Backend curriculum projects from the Holberton School Web Stack specialization.
Each subdirectory is a self-contained module with its own README, sources,
and tests.

## Modules

| Module | Topic |
| --- | --- |
| [ES6_basic](ES6_basic/) | Modern JavaScript features (ES6) |
| [ES6_classes](ES6_classes/) | Classes, inheritance, and metaprogramming in ES6 |
| [NoSQL](NoSQL/) | MongoDB shell and PyMongo |
| [pagination](pagination/) | Simple, hypermedia, and deletion-resilient pagination |
| [python_async_comprehension](python_async_comprehension/) | Async generators and async comprehensions |
| [python_async_function](python_async_function/) | `asyncio` coroutines and tasks |
| [python_variable_annotations](python_variable_annotations/) | Python type hints (PEP 484) |

## Stack

- Python 3.7 (PEP 8, type-annotated, documented)
- Node.js 12.x with Babel and Jest
- MongoDB 4.2

## Layout

Every module follows the same convention:

- Numbered task files (`0-...`, `1-...`, ...) implementing the exercises
- Optional `*-main.*` driver scripts to run a task in isolation
- A `README.md` describing learning objectives, requirements, and tasks

Refer to each module's README for setup and usage details.
